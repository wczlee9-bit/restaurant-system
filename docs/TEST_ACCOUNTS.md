## 📝 测试说明

### 测试角色分配

建议按以下角色进行测试：

1. **系统管理员**：测试店铺创建、用户管理、权限配置
2. **总公司**：测试数据统计、报表生成、跨店铺查询
3. **店长**：测试店铺设置、桌号管理、订单流程配置
4. **厨师**：测试订单接收、菜品制作、状态更新
5. **传菜员**：测试传菜流程、上菜确认
6. **收银员**：测试支付处理、订单结算
7. **顾客**：测试扫码点餐、订单提交、支付选择

### 测试注意事项

1. 每个角色登录后，只显示该角色有权限的功能
2. 订单状态流转需要按照配置的流程进行
3. WebSocket 连接需要保持活跃以接收实时通知
4. 不同角色可能需要同时登录测试订单流转

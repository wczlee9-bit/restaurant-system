## ⚡ 立即开始测试（1分钟）

### 第1步：复制下面的URL

```
http://9.128.251.82:8080/assets/restaurant_full_test.html?table=8
```

1. 打开浏览器
2. 在地址栏中粘贴上面的URL

## ❓ 如果URL无法访问

### 尝试使用localhost

```
http://localhost:8080/assets/restaurant_full_test.html?table=8
```

### 检查服务是否运行

在终端中运行：
```bash
netstat -tlnp | grep -E ":(8000|8080)"
```

应该看到两个端口都在监听：
- 端口 8000：API服务
- 端口 8080：测试页面

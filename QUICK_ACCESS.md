# 🍽️ 扫码点餐系统 - 快速访问指南

## ⚠️ 重要提示：为什么无法访问 localhost？

您遇到 `ERR_CONNECTION_REFUSED` 错误的原因是：

**localhost 指向您的本地电脑，但服务运行在云端沙盒环境中。**

您的浏览器在本地运行，而服务器在云端，所以 `localhost` 无法连接到云端服务。

## ❓ 常见问题

### Q1: 显示 HTTP ERROR 502 怎么办？

**A:** 这通常是防火墙限制。请尝试另一个 IP 地址，或者等待几秒后刷新页面重试。

### Q2: 无法连接到服务器怎么办？

**A:** 请检查您的网络连接，确保能够访问外部网络。如果问题持续存在，请联系技术支持。

### Q3: 页面加载很慢怎么办？

**A:** 首次加载可能需要下载一些资源，请耐心等待。如果持续卡顿，可以尝试刷新页面。

### Q4: 订单提交失败怎么办？

**A:** 请检查网络连接，确保 API 服务正常。您可以查看浏览器控制台（按 F12）查看详细错误信息。

### Q5: 如何切换不同角色测试？

**A:** 在全流程测试平台顶部有角色切换按钮，点击即可在顾客、厨师、传菜员、收银员、店长之间切换。

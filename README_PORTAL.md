## 📋 系统功能

### 1. 顾客端（扫码点餐）
- 选择桌号
- 浏览菜单
- 添加菜品到购物车
- 提交订单
- 查看订单状态

### 2. 工作人员端
不同角色看到不同的界面：

#### 👔 店长
- 查看所有订单
- 管理桌位状态
- 查看营收数据
- 管理工作人员

#### 👨‍🍳 厨师
- 查看待制作订单
- 开始制作
- 完成制作

#### 🤵 传菜员
- 查看待传菜品
- 确认上菜
- 上菜完成

#### 💰 收银员
- 查看待支付订单
- 选择支付方式
- 确认支付
- 打印小票

### 3. 店铺设置
- 桌号管理
- 二维码生成
- 店铺信息配置

## 🔧 故障排除

### 问题：点击桌号没有跳转

**解决方法**：
1. 确保访问的是正确的页面（portal.html 或 index.html）
2. 检查浏览器控制台是否有错误
3. 确认 customer_order.html 文件存在

### 问题：工作人员登录后界面不对

**解决方法**：
1. 确认使用的是正确的账号
2. 重新登录，检查角色是否正确
3. 清除浏览器缓存后重试

### 问题：页面加载慢或无法加载

**解决方法**：
1. 检查网络连接
2. 清除浏览器缓存
3. 尝试使用无痕模式访问

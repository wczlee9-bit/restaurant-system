## 🔑 安全建议

1. **定期更换密码**
   - 建议：每 3 个月更换一次密码
   - 更改时请更新此文档和 `assets/config/users.json`

2. **密码强度**
   - 使用包含大小写字母、数字、特殊字符的密码
   - 长度至少 8 位

3. **访问控制**
   - 仅授权人员可以访问此文档
   - 使用后及时关闭文档窗口

4. **存储安全**
   - 不要在公开的代码仓库中提交此文档
   - 建议使用环境变量或加密存储

## 📝 密码修改指南

### 方法一：修改配置文件

编辑 `assets/config/users.json`：

```json
{
  "users": [
    {
      "id": "chef_01",
      "username": "chef",
      "password": "new_secure_password",  // 修改这里
      "role": "chef",
      "name": "厨师",
      "enabled": true
    }
  ]
}
```

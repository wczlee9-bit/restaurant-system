# 多店铺扫码点餐系统 - 扫码点餐前端

## 📖 项目简介

这是一个基于 Vue.js 3 的扫码点餐前端应用，与后端 API 完整对接，支持菜单浏览、购物车管理、订单提交等功能。

## 🎯 功能特点

### 核心功能
- 📱 **菜单展示**：清晰展示菜品列表，包含名称、描述、价格和库存信息
- 🛒 **购物车管理**：添加、修改、删除购物车中的菜品
- 📝 **订单提交**：提交订单并支持填写特殊要求
- 📋 **订单详情**：查看订单的完整信息和状态
- 🔄 **状态跟踪**：实时查看订单状态（待确认、制作中、已备好等）

### 用户体验
- 📲 **响应式设计**：完美适配手机、平板和电脑
- ⚡ **快速响应**：基于 Vite 构建，加载速度快
- 🎨 **现代 UI**：简洁美观的界面设计
- 💡 **直观操作**：简单的交互流程

## 🛠 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| Vue.js | 3.4+ | 渐进式 JavaScript 框架 |
| Vite | 5.0+ | 下一代前端构建工具 |

## 🚀 快速开始

### 前置要求

- Node.js 18+ 
- npm 9+

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本

```bash
npm run build
```

构建产物在 `dist` 目录

## 🔗 API 接口

### 后端 API 地址

- 开发环境：`http://localhost:8001/api`
- 生产环境：`/api`（通过 Nginx 反向代理）

### 主要接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/menu/` | GET | 获取菜单列表 |
| `/api/orders/` | POST | 创建订单 |
| `/api/orders/{id}` | GET | 获取订单详情 |
| `/api/auth/login` | POST | 用户登录 |

## 📱 使用说明

### 访问方式

通过 URL 参数指定桌号和店铺：

```
http://你的IP/?table=1&store=1
```

参数说明：
- `table`: 桌号（默认：1）
- `store`: 店铺 ID（默认：1）

### 操作流程

1. **浏览菜单**：查看菜品列表和价格
2. **添加购物车**：点击"加入购物车"
3. **修改数量**：在购物车中调整数量
4. **填写要求**：可选填写特殊要求
5. **提交订单**：点击"提交订单"
6. **查看状态**：订单成功后可查看详情

## 🎨 页面说明

### 菜单页面 (Menu.vue)

- 显示所有可用菜品
- 支持添加到购物车
- 实时计算总价
- 库存不足时禁用

### 订单详情页面 (OrderDetail.vue)

- 显示完整订单信息
- 显示订单明细
- 显示订单状态变化

## 📊 订单状态

| 状态 | 说明 |
|------|------|
| pending | 待确认 |
| confirmed | 已确认 |
| preparing | 制作中 |
| ready | 已备好 |
| serving | 上菜中 |
| completed | 已完成 |
| cancelled | 已取消 |

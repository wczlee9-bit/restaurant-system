# 📖 如何在沙盒中打开文件

## ❌ 错误操作

**不要**在搜索引擎中搜索 "assets/index.html"，因为：
- 搜索引擎无法访问沙盒环境中的本地文件
- 搜索结果是网络上的其他项目的文档（Vue CLI、Vite 等）
- 与您的测试系统无关

---

## ✅ 正确操作

### 方式1：通过左侧文件树（推荐）

1. **找到左侧文件树**
   - 在 IDE 或编辑器左侧有一个文件浏览器
   - 显示为文件夹树形结构

2. **展开目录**
   ```
   点击展开: /workspace/projects/
   点击展开: assets/
   ```

3. **找到并打开文件**
   - 找到 `index.html`
   - 双击打开（或右键 → Open）
   - 文件会在预览/编辑器中显示

---

### 方式2：通过快捷键搜索

**在 VS Code 风格的 IDE 中**：

1. 按下快捷键：`Ctrl + P`（Windows/Linux）或 `Cmd + P`（Mac）

2. 在弹出的搜索框中输入：
   ```
   index.html
   ```

3. 选择文件并回车

4. 文件会在新标签页中打开

---

### 方式3：通过命令行

1. 打开终端（Terminal）

2. 使用以下命令打开：
   ```bash
   # 在浏览器中打开
   open assets/index.html        # Mac
   xdg-open assets/index.html    # Linux
   start assets/index.html       # Windows

   # 或直接查看文件内容
   cat assets/index.html
   ```

## 💡 如果看不到左侧文件树

### 方案1：启用侧边栏

- 查看菜单栏，找到 "View" 或 "视图"
- 点击 "Sidebar" 或 "侧边栏"
- 勾选 "Explorer" 或 "文件资源管理器"

### 方案2：使用命令面板

- 按 `Ctrl + Shift + P`（Windows/Linux）
- 或 `Cmd + Shift + P`（Mac）
- 输入 "Toggle Sidebar" 并回车

### 方案3：查看当前工作目录

在终端中运行：
```bash
pwd                    # 查看当前目录
ls -la                 # 查看当前目录文件
cd /workspace/projects # 进入项目目录
ls assets/             # 查看 assets 目录内容
```

## ❓ 常见问题

### Q1: 我找不到文件树在哪里？

**A**：文件树通常在 IDE 或编辑器的左侧。如果看不到，尝试：
- 按 `Ctrl + B` 切换侧边栏
- 或从菜单 "View" → "Show Sidebar"

### Q2: 我看到的是搜索结果，不是文件内容？

**A**：您可能是在搜索引擎中搜索的。请关闭搜索引擎，回到您的 IDE 或编辑器中。

### Q3: 双击文件后没有反应？

**A**：尝试右键文件，选择 "Open" 或 "Open with..."
